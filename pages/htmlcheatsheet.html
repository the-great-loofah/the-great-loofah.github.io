HTML Cheat Sheet
HTML Page Setup: To Link Javascript: script src = "Name of JS file.js" closing script tag.
Add "defer" after the script source if accpeting input. Skips the DOMcontentloaded in JS.
To link Css: link rel = "stylesheet" href="Name of CSS file.css" closing link tag
Text Input: form id="custom form name"
input type:"(text) (number) (email) (etc)"
button id="custom ID" button text
Function Call Setup: inside of whatever tag "(activate parameter) + (functionname())"
Div Template: Assign Class for CSS and ID for JS, both can be the same for organization.
Assigning display types in CSS should target the parent div, not the content inside.
Image Setup: (self closing) src="internet link or directory" alt="text if image fails to load"
Misc Items: table: Requires tr for row and td for columns
legend: Placed inside a fieldset.
dl: dt for item to be described, dd indents and in smaller texts for description. Good for indents in general
ul: needs li, produces an indented list with bulletpoints
ol: needs li, produces an indented list numbered in order
textarea: same as input but can increase its row and column span
Link different HTML pages: anchor tag href="destination" text for the link
Javascript Cheat Sheet
Javascript Page Setup: If collecting input, use 'DOMContentLoaded or defer in HTML script source
Variables: let "variable name"
let "variable name" = document.getelementby(id) or (class)
Convert HTML input to javascript useable "variable name".value
Function: function "name of function" "(any parameters being used)"
Collect Input from HTML: addEventListener("event to listen for"), function(event) {"what happens when"};
Loops: for: loops until variable value reached
while: loops while variable within a certain value
Arrays "variable name" ["array content", "another array content"]
foreach: Loops through the array then stops
Objects: const "Name of object" {"Parameter 1": "string", "parameter 2:" number, etc...}
for in loop. for (const i in "name of object") loops through properties of an object.
Constructors: function "name of constructor"("parameter 1, parameter 2...") {this."parameter 1" = "paramter 1"}
Use Constructor: const "object name" = new "name of constructor"("value for parameter 1..." etc.)
Checking Values: console.log("name of variable/object")
Changing HTML with Javascript: document.getElementByID("id of element").innerHTML = "text"
document.write("text") replaces ALL HTML.
Creating Elements With Javascript: document.createElement("name of element")
setattribute("attribute", "value")
"location".appendchild.("elements to be created")
